# Security Advisory: SQL Injection Vulnerabilities in Online Shopping System PHP v1.0

This repository contains the public advisory for multiple SQL Injection vulnerabilities discovered in the **online-shopping-system-php v1.0** software.  
These vulnerabilities were identified during security research and have been assigned CVE identifiers which are now ready for publication and verification via the MITRE CVE Program.

## Affected Software

- **Product Name:** Online Shopping System (PHP)
- **Version:** v1.0
- **Repository:** https://github.com/indieka900/online-shopping-system-php

## Assigned CVE IDs

The following CVE identifiers were reserved for distinct SQL Injection issues:

- **CVE-2025-61320**
- **CVE-2025-61321**
- **CVE-2025-61322**
- **CVE-2025-61323**
- **CVE-2025-61324**

These CVE IDs are currently in *Reserved* status and pending publication after this advisory link is submitted to MITRE‚Äôs CVE Request Form.

## Summary of Vulnerabilities

Each of the listed CVEs corresponds to a specific SQL Injection vulnerability found in the application. The vulnerabilities allow an attacker to manipulate SQL queries through unsanitized input parameters, potentially leading to unauthorized data access, data exfiltration, or other security impacts.

Full details, including affected files, parameters, impact descriptions, proof-of-concepts (PoCs), and suggested remediation steps are available in this repository.
